## 搜索技能

### `/news` - 新闻搜索

**功能**: 获取最新新闻和热点信息

**使用方法**:
```
/news              # 获取今日头条热榜
/news <关键词>     # 搜索包含关键词的新闻
```

**示例**:
```
用户: /news
Bot: [显示今日头条热榜前 15 条]

用户: /news 人工智能
Bot: [显示人工智能相关新闻]
```

**实现位置**: `internal/command/router.go:296-333`
**搜索引擎**: 今日头条热榜（主）、Bing News（备）

**使用建议**:
- 用户询问"今天的新闻"、"最新新闻"、"热点" → 告诉用户使用 `/news`
- 用户询问特定主题新闻 → 告诉用户使用 `/news 关键词`
- 搜索结果包含标题、链接、热度等信息

## 对话管理技能

### `/ask` - 向 Claude 提问

**功能**: 向 Claude 提问，续接当前会话上下文

**使用方法**:
```
/ask <问题>
或直接发送消息（等同于 /ask）
```

**特性**:
- ✅ 自动记忆上下文（保存在 session 中）
- ✅ 支持图片识别（发送图片 + 文字）
- ✅ 自动模型选择（根据任务复杂度）
- ✅ Token 使用统计

**实现位置**: `internal/command/router.go:99-204`

---

### `/new` - 开启新对话

**功能**: 清除当前会话上下文，开始全新对话

**使用方法**:
```
/new
```

**场景**:
- 用户想切换话题
- 当前对话太长，需要清理上下文
- 用户明确说"重新开始"、"新对话"

**实现位置**: `internal/command/router.go:208-216`

## 记忆管理技能

### `/remember` - 保存长期记忆

**功能**: 保存用户偏好、个人信息等长期记忆，每次对话自动注入

**使用方法**:
```
/remember <内容>
```

**示例**:
```
用户: /remember 我喜欢简洁的回答，不要太啰嗦
用户: /remember 我是 Python 开发者，工作在北京

[后续对话会自动注入这些记忆]
```

**实现位置**: `internal/command/router.go:220-231`

## 系统工具

### `/model` - 模型选择

**功能**: 查看或设置 Claude 模型偏好

**可用模型**:
- `opus` - Opus 4.6（最强，适合复杂任务）
- `sonnet` - Sonnet 4.5（平衡，适合大多数任务）
- `haiku` - Haiku 4.0（快速，适合简单任务）

**使用方法**:
```
/model              # 查看当前模型
/model <模型名>     # 设置模型偏好
```

**注意**: 设置偏好后，系统仍会根据任务复杂度自动选择最优模型

## 🎯 Agent 使用指南

### 何时引导用户使用技能

1. **搜索类请求**:
   - 用户问："今天有什么新闻？" → 告诉用户使用 `/news`
   - 用户问："搜索一下 XXX 新闻" → 告诉用户使用 `/news XXX`

2. **对话管理**:
   - 用户说："切换话题" → 告诉用户使用 `/new`
   - 用户说："查看历史" → 告诉用户使用 `/history`

3. **记忆管理**:
   - 用户说："记住，我喜欢..." → 告诉用户使用 `/remember`
   - 用户说："忘掉之前说的" → 告诉用户使用 `/forget`

### 回复模板

```
用户: 今天有什么新闻？
Bot: 你可以使用 `/news` 命令查看今日头条热榜，或者使用 `/news 关键词` 搜索特定主题的新闻。

用户: 我想切换话题
Bot: 使用 `/new` 命令可以清除当前对话上下文，开始新的对话。

用户: 记住我喜欢简洁的回答
Bot: 好的！你可以使用 `/remember 我喜欢简洁的回答` 来保存这个偏好，之后每次对话都会记住。
```

## 🔧 技术实现参考

### 命令路由
- **路由器**: `internal/command/router.go`
- **处理器**: 每个命令对应一个 `Handler` 接口实现
- **注册**: 在 `NewRouter()` 中注册所有命令

### 数据存储
- **会话**: `memory.Store.SaveSession()`
- **历史**: `memory.Store.SaveHistory()`
- **记忆**: `memory.Store.AddMemory()`
- **使用量**: `memory.Store.RecordUsage()`

### 搜索实现
- **搜索器**: `internal/newsearch/searcher.go`
- **引擎**: 今日头条（主）、Bing（备）
- **格式化**: `newsearch.FormatNewsItems()`
